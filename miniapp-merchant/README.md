# 商家端微信小程序联调骨架

这个目录是商家端小程序第一版联调骨架，目标是让门店现场能先跑通核心运营动作。

已覆盖页面：

1. 工作台：订单、AI试发、AI转化、成交金额概览。
2. 订单：按状态查看订单，确认预约、确认到店、开始服务、完成服务、取消订单。
3. AI次数：给顾客赠送1次AI试发，给主理人追加当天赠送额度，查看赠送转化。
4. 业绩：门店成交、AI转化成交、主理人排行、服务项目统计。

## 打开方式

用微信开发者工具导入 `miniapp-merchant` 目录。

本地后端默认地址在 `app.js`：

```js
apiBaseUrl: "http://127.0.0.1:8000"
```

演示默认身份：

```js
tenantId: 1
storeId: 1
staffId: 2
```

正式上线后，`tenantId/storeId/staffId` 应来自商家端登录态和权限接口，不应写死。
